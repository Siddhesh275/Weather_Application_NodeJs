# Weather_Application_NodeJs

## Built with

### Front end
* HTML 
* CSS
* Bootstrap

### Backend
* Nodejs
* ExpressJS

## Inital setup
 1. Clone github repository or Download the zip file

2. Install all dependencies using command
```
npm install
```
3. Start server using command
```bash 
node app.js
```
